# 1. npm

### Node Package Manager

- 다른 사람들이 만든 소스 코드들을 모아둔 저장소
- 오픈 소스 생태계를 구성중
- 바퀴를 다시 만드려고 하지마라
- 패키지 : npm에 업로드된 노드 모듈
- 의존관계라고 부름

## 1. package.json

### 현재 프로젝트에 대한 정보와 사용중인 패키지에 대한 정보를 담은 파일

- 같은 패키지여도 버전별로 기능이 다를 수 있어서 버전을 기록해둠
- 동일한 버전을 설치하지 않으면 문제가 생길 수 있음
- 노드 프로젝트 시작 전 package.json부터 만들고 시작

```shell
npm init
```
